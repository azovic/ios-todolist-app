# 📱 iOS Todo List App

A clean and minimal **Todo List application** built with **Swift** and **SwiftUI**.
This app helps users organize their daily tasks with a simple and intuitive interface.

---

## 🚀 Features

* ➕ Add new tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* 🔍 Filter tasks (All / Active / Completed)
* 📋 Clean and modern UI

---

## 🛠️ Technologies Used

* Swift
* SwiftUI
* Xcode

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/azovic/ios-todolist-app.git
```

2. Open the project:

```bash
open ios-todolist-app.xcodeproj
```

3. Run the app using Xcode Simulator
